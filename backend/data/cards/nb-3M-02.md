# nb-3M-02

START
Complete the log-space Naive Bayes formula:

ŷ = argmax_c [  +  ]
END

START
ŷ = argmax_c [ log P(y = c) + ∑ᵢ log P(xᵢ | y = c) ]

First blank: log P(y = c) (log prior)
Second blank: ∑ᵢ log P(xᵢ | y = c) (sum of log likelihoods)

Product becomes sum in log space.
END
