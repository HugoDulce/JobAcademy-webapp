# nb-1C-01

START
When is Naive Bayes a good choice for classification?
END

START
High-dimensional sparse data where features are roughly independent given the class.<br><br>Best for (Uses):<br>- Text classification -> (1. spam, 2. sentiment)<br>- Document categorization<br>- Real-time prediction (extremely fast)<br>- Baseline models<br><br>Not for:<br>- Feature interactions matter<br>- Need calibrated probabilities<br>- Continuous features with complex distributions
END
