# nb-1C-01

START
When is Naive Bayes a good choice for classification?
END

START
High-dimensional sparse data where features are roughly independent given the class.

Best for:
- Text classification (spam, sentiment)
- Document categorization
- Real-time prediction (extremely fast)
- Baseline models

Not for:
- Feature interactions matter
- Need calibrated probabilities
- Continuous features with complex distributions
END
