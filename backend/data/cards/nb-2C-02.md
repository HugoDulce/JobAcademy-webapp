# nb-2C-02

START
How do you choose which Naive Bayes variant to use?
END

START
Match the variant to your feature types:

GaussianNB: Continuous features, assume normal distribution
MultinomialNB: Count features (word frequencies, term counts)
BernoulliNB: Binary features (word presence/absence)
ComplementNB: Imbalanced datasets with multinomial features

Most common: Gaussian for numeric data, Multinomial for text.
END
