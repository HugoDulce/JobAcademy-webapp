# nb-2M-01

START
Given:
P(x₁=1, x₂=1 | y=A) = 0.12
P(x₁=1 | y=A) = 0.3
P(x₂=1 | y=A) = 0.5

Are x₁, x₂ independent given y=A?
END

START
Test: P(x₁, x₂ | y) = P(x₁ | y) × P(x₂ | y)?

Left side: 0.12 (given)
Right side: 0.3 × 0.5 = 0.15

0.12 ≠ 0.15 → NOT independent

Interpretation: Features correlate within class A. NB's assumption is violated. Model will be miscalibrated.
END
